# OurBooks
OurBooks clinet build by TEAM YUML

## INSTALLATION

### Prerequisites
```
Node 10
```

### JAVASCRIPT
#### install dependencies
```
npm install
```

#### To start dev server use:
```
npm run dev
```

#### build for production and view the bundle analyzer report
```
npm run build --report
```

### RUNNING TESTS
#### run all tests
```
npm run test
```
